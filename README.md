# 🧠 Buddy

### The memory that makes your AI **smarter the more it remembers — not slower, not costlier.**

Buddy is a memory and a way-of-working you install on your AI agent. It learns your world,
remembers it across every session, and — unlike most memory systems — **won't confidently
remember things wrong.**

Bring your own model. Plain text files, git-friendly, no database. Free (MIT).

---

## The problem with AI memory today

Most AI memory gets *worse* as it grows. The more it remembers, the more it has to cram into
its head every time it answers — so it gets **slower and more expensive** exactly when it should
be getting smarter. And when a new fact arrives, it usually just overwrites the old one — so over
time it quietly starts **remembering things wrong.**

## What Buddy does differently

- **🚀 Smarter, not costlier.** Buddy's memory is a tiny table-of-contents that points to
  one-page notes. However big the brain grows, Buddy only ever opens the contents page plus the
  two or three notes it actually needs. A brain with 60 notes and one with 60,000 cost about the
  same to use. *More memory = more intelligence, at a flat cost.*
- **🛡️ Never remembers wrong.** When a new fact seems to clash with an old one, Buddy doesn't
  silently overwrite — it usually means the two apply in *different situations*. Buddy keeps both
  (noting when each applies) or asks you. A true fact is never lost.
- **✨ A sharp brain, not a junk drawer.** Buddy only saves what's worth keeping — reusable,
  non-obvious, hard-won. The brain stays signal, not noise.
- **👥 It knows your people.** Everyone you work with gets one card — the single source of truth —
  so Buddy never re-asks what it already learned about a client or contact.

## How it works

Install Buddy like setting up a new phone: it asks you a few quick questions, then builds *your*
brain — your folders, your style, your starter card. From then on it remembers your preferences,
projects, and people across every session. No re-explaining, ever.

Everything is plain Markdown in one folder you own. You can read it, edit it, back it up, or
commit it to git. Nothing is locked in a database.

## Install (about 2 minutes)

> v1 runs on **Claude Code**. More apps coming.

1. **Get Buddy** — download or `git clone` this into a new folder. That folder is your workspace.
2. **Open it in Claude Code** — point Claude Code at that folder.
3. **Answer 5 questions** — Buddy introduces itself and sets itself up.
4. **Start working** — Buddy now remembers everything, right there in that folder.

That's it. No database, no extra setup beyond the AI model you already use.

> One thing to remember: **always start Buddy in this same folder** — that's where its memory
> lives. Buddy makes and sorts every file inside it for you; you never organize anything.

## Getting the most out of Buddy

A few habits that make Buddy noticeably smarter over time:

- **Always open Buddy in the same folder.** That's its memory — the single most important habit.
- **Answer its save prompts.** Buddy asks *"worth saving? (ok/skip)"* — a quick yes or no keeps its brain sharp.
- **Correct it out loud.** The moment you say "actually, it's X," Buddy offers to remember it. Corrections are how it gets smarter.
- **Introduce your people once.** Tell Buddy about a client or contact, and it won't re-ask next time.
- **Say "tighten"** if a reply runs long — Buddy cuts it on the spot.
- **Now and then, say "reconcile the brain"** — Buddy tidies up, merges duplicates, and flags anything confusing.
- **Let Buddy keep the house.** Don't reorganize the files by hand — Buddy sorts them so the memory keeps working.

## What you get

A complete second brain: a five-shelf knowledge library (fixes, how-tos, rules, facts, skills),
people & client cards, day-to-day memory, and a built-in habit of saving what it learns and
tidying itself up — all curated with your approval.

## Why it's called Buddy

Buddy is my son's nickname. Watching him learn a little more every day — and never forget the
people who matter to him — is what I wanted to put into software. So Buddy grows up the same way:
it gets to know your world, holds onto what matters, and becomes more *yours* the longer you spend
together. I hope it looks after you the way I hope the world looks after him.
